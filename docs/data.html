Then open chandniverse.github.io.Rproj in RStudio.

### Package Versions

```{r}
#| label: pkg-versions
#| echo: false
#| message: false

pkgs <- c("ggplot2", "plotly", "dplyr", "knitr")
installed <- sapply(pkgs, function(x) {
  if (requireNamespace(x, quietly = TRUE)) {
    as.character(packageVersion(x))
  } else {
    "Not installed"
  }
})

data.frame(
  Package = pkgs,
  Version = installed
) |> knitr::kable()
```

## How to Reproduce

### Option 1: Clone the Repository

```bash
git clone https://github.com/chandniverse/chandniverse.github.io.git
cd chandniverse.github.io
```

Then open `chandniverse.github.io.Rproj` in RStudio.

### Option 2: Download Individual Files

Download the specific scripts and data files you need from the tables above.

### Option 3: Run in R

```r
# Install required packages
install.packages(c("ggplot2", "plotly", "dplyr", "knitr"))

# Source the analysis script
source("scripts/analysis.R")
```
